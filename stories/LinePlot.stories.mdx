# LinePlot

Standard line plot to accomodate multiple data sets.

<Props of={LinePlot} />

## Multi Line Plot

Hover over a label to highlight the plot of interest. Hover over a line to highlight tooltips for each point.

<Preview>
  <Story name="multiple" height="550px">
    {{
      components: { LinePlot },
      template: `
        <div>
             <LinePlot
              class="w-75"
              :xValues="['1', '2', '3', '4', '5', '6', '7']"
              :yValues="[['2', '4', '3', '1', '2', '1', '2'],
                         ['3','6', '2', '1', '0','4', '3'], 
                         ['9','5', '8', '5', '3','2', '1'],
                         ['5','4', '6', '3', '2','3', '6'],
                         ['4','5', '6', '7', '5','4', '7']]"
              :yLabels="['Alpha', 'Bravo', 'Charlie', 'Echo', 'Delta']"
              />
        </div>
        `
    }}
  </Story>
</Preview>

## Dual Axis Line Plot

A dual axis requires that there are only two sets of yvalues, where the first set is scaled on the left and the second set is scaled on the right.

<Preview>
  <Story name="dual axis" height="550px">
    {{
      components: { LinePlot },
      template: `
        <div>
             <LinePlot
              class="w-75"
              dualAxis
              :xValues="['1', '2', '3', '4', '5', '6', '7']"
              :yValues="[['1.1', '1.2', '1.0', '0.9', '0.85', '0.75', '0.7'],
                         ['0.2','0.3', '0.4', '0.8', '0.9','0.9', '0.95']]"
              :yLabels="['First', 'Second']"
              />
        </div>
        `
    }}
  </Story>
</Preview>
